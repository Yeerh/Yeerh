### BeautyFlow

Aplicação para agendamento de serviços em barbearias e salões de beleza.

O projeto possui landing page comercial, área do cliente, calendário de agendamentos, autenticação com Google, JWT e integração com banco de dados.

**Tecnologias:** React, TypeScript, Tailwind CSS, Express, Prisma e PostgreSQL.
